# 🚀 Social Media Platform Backend

A robust and scalable backend system for a social media platform, developed as part of a PEP (Project Execution Program). This project is built using Spring Boot, PostgreSQL, and Supabase.

## 📌 Features

- User Authentication (JWT-based)
- User Profile Management
- Post Creation & Management
- Like & Comment System
- Follow / Unfollow Users
- Supabase PostgreSQL Integration

## 🛠️ Tech Stack

- Spring Boot
- Java
- PostgreSQL (Supabase)
- Hibernate / JPA
- JWT Authentication
- Maven

## ⚙️ Setup Instructions

1. Clone the repository:
   git clone https://github.com/your-username/social-media-backend.git

2. Configure database in application.properties

3. Run the project:
   mvn spring-boot:run

## 📡 API Endpoints

- POST /api/auth/register
- POST /api/auth/login
- GET /api/posts
- POST /api/posts
- POST /api/posts/{id}/like
- POST /api/posts/{id}/comment

## 🔒 Security

- JWT Authentication
- BCrypt Password Encryption

## 🧪 Testing
